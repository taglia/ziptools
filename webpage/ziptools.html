ZipTools for Linux

This little program can control a zip drive without having to type long commands. I've taken Guenther's "ziptool" and built a Qt graphic interface around it, with some extra features. I've tested it only with my parallel drive, but I think it should work with the SCSI version, too. The code is almost totally uncommented, but by now I have not had any time to comment it.

Features

  • Change protection mode: It supports all of the zip protection modes:
    • Read Write;
    • Write protected;
    • Write protected with password;
    • Read/Write protected with password (with a recent kernel).
  • Partition and format a disk: it can format a disk using the ext2 file-system or with the fat. To partition a disk the program needs the "sfdisk" command, and to make the file-system it needs "mke2fs" and "mkdosfs". When formatting a disk with fat, the program uses the 4th partition; this is done to let the formatted disk be read by Macintosh computers, too.

Requirements

  • A Linux machine with a working X environment;
  • The Troll Tech Qt libraries;
  • A parallel (tested) or SCSI (untested) zip drive.
